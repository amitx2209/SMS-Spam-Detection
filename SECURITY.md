# Security Policy

## Supported Versions

This project is actively maintained on the `main` branch.

Older versions, experimental notebooks, or demonstration artifacts (screenshots, videos) are not guaranteed to receive security updates.

---

## Intended Use

This project is intended for:

• Academic learning  
• Demonstration of machine learning concepts  
• Local experimentation and portfolio usage  

It is **not intended for production-grade or mission-critical deployment**.

---

## Reporting a Vulnerability

If you discover a potential security issue, please do not disclose it publicly.

Instead, report it responsibly by:

• Opening a GitHub issue with minimal details  
• Clearly describing the vulnerability and reproduction steps  

Avoid including sensitive data, credentials, or personal information.

---

## Responsible Disclosure

We appreciate responsible disclosure of security concerns.

Valid reports will be reviewed, and appropriate corrective action will be taken when feasible.

---

## Limitations

• The application does not process real SMS inbox data  
• No personal user data is stored or transmitted  
• Demo screenshots and videos are for visualization purposes only  

Users are responsible for ensuring safe usage in their local environment.
